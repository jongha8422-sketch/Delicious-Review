# Delicious-Review
現代人は外食を頻繁にしますが、グルメ情報や口コミを探すのが難しく、
SNSやブログの情報は整理されていないため信頼性が低いです。そこで、
予約とレビューを統合したシステムを作り、
情報の確認や評価を一カ所で行えるようにします。
